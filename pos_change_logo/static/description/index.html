POS Change Logo

POS Change Logo module helps you to change the logo of Point Of Sale. Using this module you can define a Custom Logo for Point Of Sale.The custom logo will appear on your POS Screen and can be used for marketing purpose and also helps in brand promotion. For example, you have a POS store and you want to change the logo of the store, this can be done by using our module. POS Change Logo module is relevant to target audience.

Features
  • It allows the pos user to set custom logo for the Point Of Sale. User can select logo in the corresponding POS configuration.
  • Selected logo will automatically be shown in the POS session.
  • Selected logo will be shown in both POS Ticket or Receipt(Xml Receipt)
  • Session View & Configuration
