Freight Transport Management and Delivery Routes Odoo Apps

This Odoo apps helps to manage Freight Transportation Management and Logistics System which has key features which include transporter details,Vehicle information, route info, LR number of parcel all delivery order details. This create auto entry transport entry with reporting those entries make it more easy to operate Transport details in Odoo app. In Odoo managing Trans-shipment and vehicle for transporter become super easier.

Features

User can see transport details

User can see and add transport details in app which include all the details.

Transport rout details

User can see and add rout details, for better record.

Sale order got details of transport.

User can see details like parcel no, rout, transport via etc in the sale order.

Report contains details.

Report contains all the details about transport.

Configuration

You have to set transport details in system to set that details there is menu under Sales > Configuration > Transporters Details.

Transporter Details are shown below.

Transporters Routes Details

You can set the details of the transporters routes with Source and Destination Location.

You can set the details of Transporters Routes Locations

Assigning transport details to sale order

While creating new quotation you can assign transport via to state , this order will be transfer via this transporters.

Confirming Sales order

Confirming sale order that will pass transport details in picking.

Transport Route Locations entry is set based on route.

You can also see related transport entry for the picking.

Entering no of parcels wizard

You can also enter no of parcels in picking through the button on picking.

One Transport Entry will be generated while confirming the picking.


To view how many transport entry are their you can see the view of transport entry under Sales > Transport entry menu.

New transport entry will in draft stage if user want to hold it can be done through halt button.

Once transport entry goes in waiting state two options are there you can done it or you can cancel it.

If you cancel it there will be another option to reschedule the transport entry it will creates the new entry for the same delivery order in draft state.
