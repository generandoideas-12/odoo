Import and Update Product and Contact(Customer/Supplier/vendor) from Excel or CSV File in odoo

If you have bulk of products or partner in excel or csv format. When you get data of partner or products from marketing company then don't need to enter them manually in odoo. With using this app you can import them quickly in odoo.

Features

User can import product.

User are allowed to import product or update product.

User can import partner

User are allowed to import partner or update partner.

Errors or warnings

Users can see errors it he/she tries to import wrong salesperson, category etc or wrong date file.

User can import data in bulk

User are allowed to import data in bulk.
